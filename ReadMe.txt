Important information notes from Nathan:
Display View Distance is based on a mix of Render Distance and distance. for this case, we will assume your Entity Distance is set to 100%. 

A View Range of 1 starts at a distance of 64 blocks by default, and this number is static until it reaches beyond the threshold of 8 render distance.

Each proceeding number in render distance increases the DEFAULT distance it can be viewed at by 8 blocks, rather than increasing the MAXIMUM distance. On a personal note, I find this rather ridiculous as it makes it impossible for a uniform method of optimization for all players, so it's best to stick with the assumption of 8 Render Distance.

Display Entity Distance Default:
1=64 blocks

Maximum Distance in Vanilla:
1=80 blocks

Fabric Mods Distance Maximum:
1=160 blocks (HARD LIMIT)

It is for these reasons that it is HIGHLY RECOMMENDED you have your render distance at 8 (or lower, if nessasary) for setup.


I'm noting these because I believe it is critical that correct information is known, as many players have conflicting information about the range, one even suggesting it's based on 160 blocks. I have yet to determine what mod specifically at this time, but I know it's a mod I myself also have installed


Finally, Keep in mind that jumping can greatly affect range. always double check your setup by jumping. If you have plugins that allow for player scaling or crawling, these can also affect range, so be sure to check these too.
